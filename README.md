# Pashu-Aahar

Pashu-Aahar is a smart cattle nutrition Android application designed to help farmers generate balanced feed recommendations based on cow breed, milk yield, region, temperature, and season. The app also provides weather analysis, veterinary tips, and cost tracking to improve cattle health and reduce feeding expenses.

---

## Features

* Google Sign-In Authentication
* Smart Feed Recommendation System
* Cow Profile Management
* Weather & Temperature Analysis
* Veterinary Tips
* Feed Cost Tracker
* Saved Feed Plans
* Offline Support using Room Database

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Pashu-Aahar.git
```

2. Open project in Android Studio

3. Sync Gradle

4. Run the app using Android Emulator or physical device
